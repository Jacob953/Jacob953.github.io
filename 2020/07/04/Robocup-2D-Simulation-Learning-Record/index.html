配置robocup 2D simulation环境

注意事项：

​ 1.start.sh实际上就是几个包的安装操作（如果没有的同学请自取）

​ 2.想要将Windows直接导入VMware需要安装VMwareTools(或者不移除USB接口，采用USB导入，这样可能会导致文件损失)

​ 3.很多教程是以Ubuntu 12.为基础的（如上的start.sh便是），部分操作无法正常进行，我将会一一指正

​ 4.Ubuntu在终端输入密码时，光标是不会移动的，只管输就行（Ubuntu想要保护用户安全）

​ 5.在安装过程中，碰到[Y/n]则输入Y，其余情况全按照[...]里面的输入

安装VMware Tools

​ 1.安装VMware Tools（我装过了，就变成了“重新安装”，找不到的小白可以去管理看看）
